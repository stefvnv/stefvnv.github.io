Employee Portal
Overview

The Employee Portal desktop application serves as an efficient employee management system that seamlessly integrates a graphical user interface (GUI) with an SQL database. This application streamlines employee data management, login authentication, and various actions, making it a valuable tool for organizations of all sizes.


Features

User-Friendly Login and Dashboard

The application starts with a login page where users must enter the correct username (root) and password (pw) to gain access. Once logged in, users are greeted with a user-friendly dashboard displaying a live clock, employee details, and quick navigation buttons.


Efficient Employee Data Management

Users can view, update, and delete employee records effortlessly. Clicking on an employee ID populates the details on the right side of the interface. The application also supports live search functionality to quickly locate specific employee records.


Secure Singleton Implementation

To ensure data integrity, the application employs singleton classes for "Manage Employee" and "Add Employee" functionalities. This prevents multiple instances of these actions from opening simultaneously, enhancing data consistency.


Seamless Employee Addition

Users can add new employees with ease by filling in essential details such as name, date of birth, and a profile picture. The application guides users to upload an employee image, ensuring a comprehensive employee database.
