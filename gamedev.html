Sailor Moon: The Game

Sailor Moon: The Game is the first game I created using GameMaker Studio. This platform game consists of six levels of increasing difficulty. The aim of the game is to save the galaxy by killing as many enemies as you can, as well as the boss at the end of the game.

Click the download button below to download and play Sailor Moon: The Game.

Pixel Paint

As the second game I ever created, Pixel Paint helped me understand programming from a new, more object-orientated persepective. To make this game, I used Game Maker Language (GML). Pixel Paint is a fun puzzle game where the player must figure their way out through each level. The player must only touch the wall color that matches the current color of the player.

Pixel Paint is still under development and therefore is currently not available for download.

Touch gems to switch color

Avoid touching the wrong colored blocks

Jump on enemies to kill them

Touch jumping blocks to jump to hard to reach places
