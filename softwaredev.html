Java Guessing Game

This Java program stores a random number from 1 to 1000 as a variable 'numberToGuess' and by calling the Scanner class, allows the user to input numbers in attempt to guess the number. Through the use of a while loop, the user is given a hint after each input to know whether they should increase or decrease their guess the next time they input a number. Once the input (stored as variable 'guess') is equal to the 'numberToGuess' variable, the program will come to an end and tell the user how many 'numberOfTries' it took them to win the game.
